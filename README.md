# pdbfetch

Simple cli for downloading molecule structure files

```
Usage: fetchpdb [-format] [-region] [id1...]
  -format string
    	format of file {pdb|cif} (default "pdb")
  -region string
    	region of mirror {us|eu|jp} (default "us")
```
